MMB322 Toolbox

The MMB322-Toolbox is a application for the Android set-top box “Meteorit MMB322″ from PEARL.
More information about this product is available in the shop: http://www.pearl.de/a-PX1534-1600.shtml

The main purpose of the toolbox was to provide a easy way to root the box via network-adb.
But in later versions I added much more features like installing android-market, recovery-reboot, editing build.prop and executing adb-shell commands.

The application is written in Java and uses SWING for the user interface.
It contains native adb-clients for Linux, Mac and Windows and automatically uses the correct binary so it runs on all major operating systems.

All information about this toolbox(download, changelog, source) can be found in this german forum:
http://www.android-hilfe.de/sonstige-android-geraete/115788-mmb-322-toolbox.html
